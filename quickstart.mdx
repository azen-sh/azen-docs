## Step 2: Store your first memory

Weâ€™ll store a simple user preference.

<AccordionGroup>
  <Accordion
    icon="database"
    title="Send a memory to Azen"
    defaultOpen
  >
    Use either cURL or TypeScript:

    <Tabs>
      <Tab title="cURL">
      ```bash
      curl -X POST https://api.azen.sh/api/v1/memory \
        -H "Content-Type: application/json" \
        -H "azen-api-key: YOUR_API_KEY" \
        -d '{
          "text": "User prefers cold brew coffee",
          "dedupKey": "preference:coffee"
        }'
      ```
      </Tab>

      <Tab title="TypeScript">
      ```ts
      import fetch from "node-fetch";

      const response = await fetch("https://api.azen.sh/api/v1/memory", {
        method: "POST",
        headers: {
          "Content-Type": "application/json",
          "azen-api-key": process.env.AZEN_API_KEY as string
        },
        body: JSON.stringify({
          text: "User prefers cold brew coffee",
          dedupKey: "preference:coffee"
        })
      });

      const data: {
        ok: boolean;
        memoryId: string;
        createdAt: string;
        status: "processing";
      } = await response.json();

      console.log(data);
      ```
      </Tab>
    </Tabs>
  </Accordion>

  <Accordion icon="clock" title="What happens after this?">
    When you send a memory:

    1. Azen stores it in your private namespace  
    2. It queues the text for vector embedding  
    3. The memory becomes searchable once embedding finishes  

    You donâ€™t manage any of this infrastructure.
  </Accordion>
</AccordionGroup>

## Step 3: Search your memories

Now letâ€™s retrieve that memory using natural language.

<AccordionGroup>
  <Accordion
    icon="magnifying-glass"
    title="Search using natural language"
    defaultOpen
  >
    Use your stored memories with semantic search:

    <Tabs>
      <Tab title="cURL">
      ```bash
      curl -X POST https://api.azen.sh/api/v1/memory/search \
        -H "Content-Type: application/json" \
        -H "azen-api-key: YOUR_API_KEY" \
        -d '{
          "query": "What kind of coffee does the user like?"
        }'
      ```
      </Tab>

      <Tab title="TypeScript">
      ```ts
      import fetch from "node-fetch";

      const response = await fetch("https://api.azen.sh/api/v1/memory/search", {
        method: "POST",
        headers: {
          "Content-Type": "application/json",
          "azen-api-key": process.env.AZEN_API_KEY as string
        },
        body: JSON.stringify({
          query: "What kind of coffee does the user like?"
        })
      });

      type Memory = {
        id: string;
        content: string;
        metadata?: Record<string, any> | null;
        createdAt: string;
        embedded: boolean;
      };

      const data: {
        memories: Memory[];
      } = await response.json();

      console.log(data.memories);
      ```
      </Tab>
    </Tabs>
  </Accordion>

  <Accordion icon="brain" title="How search works under the hood">
    Azen embeds your query, compares it with your stored memories, and returns the most semantically relevant ones.

    All embeddings, indexing, and vector search are handled for you automatically.
  </Accordion>
</AccordionGroup>
